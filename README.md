# Sound Captain Gemini AI Server

Render settings:

Build Command:
```bash
npm install
```

Start Command:
```bash
npm start
```

Environment variable:
```text
GEMINI_API_KEY = your Gemini key
```

App backend URL:
```text
https://YOUR-RENDER-URL.onrender.com/api/chat
```

Test:
```text
https://YOUR-RENDER-URL.onrender.com/health
```
